Variaveis de Variaveis (Array Multidimensional)

variavel simples: Armazena apenas uma informação;

Variavel composta: Armazena mais de uma informação;

Variaveis de variaveis: Armazena varável dentro de outra variável, assim podemos organizar várias informações dentro de uma única variável.
